# Projeto_01_endless_gnome

Este é um jogo top-down desenvolvido no Godot como parte de um projeto colaborativo.

---

## Sobre o Grupo

O grupo de jogos tem como principal objetivo criar e desenvolver jogos digitais. Entretanto, sua proposta vai além da simples produção de jogos, a ideia central é formar uma comunidade colaborativa voltada à criação, experimentação e aprendizado coletivo. O grupo é composto por alunos da UniFil, abrangendo desde o primeiro até o quarto ano, o que possibilita uma integração entre diferentes níveis de conhecimento e experiência. Dentro desse ambiente, buscamos não apenas desenvolver jogos, mas também experimentar novas mecânicas, estimular a criatividade e promover o crescimento técnico e artístico dos integrantes.A proposta inclui proporcionar aos membros experiências práticas em trabalho em equipe, gestão de tempo, definição e cumprimento de metas, documentação de projetos, criação de artes e programação. Dessa forma, cada participante pode se envolver em diferentes etapas do processo de desenvolvimento de um jogo, entendendo como as áreas se interconectam.

Atualmente, o grupo está focado na criação de um primeiro projeto, que servirá como demonstração (demo) do potencial do grupo. A longo prazo, o grupo também planeja criar um fundo de investimento interno, que será alimentado conforme os jogos forem sendo lançados. Essa iniciativa visa sustentar os futuros projetos, financiar equipamentos e fortalecer a identidade do grupo como uma comunidade independente e criativa de desenvolvimento de jogos.

## Evolução do Código

### Código Antigo

- Sistema simples de movimentação
- Sem controle de ataque
- Sem inimigos
- Código mais direto , mas foi feito para estudos.

### Código Atual

- Sistema de combate ativo
- Uso de HitBox e HurtBox separados
- Cooldown de ataque
- Knockback ao receber dano
- Inimigos com comportamento (seguem e atacam o player)

---

## Resumo

O projeto evoluiu de uma base simples para um sistema mais completo e estruturado, isso graças aos esforços dos alunos.
